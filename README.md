# kompressor

Kompressor is a prototype language based on [Io Language](http://iolanguage.org/).

## Install & run

    $ git clone https://github.com/ianpreston/kompressor
    $ cd kompressor
    $ python main.py


## Hello world

    >>> x := "Hello, world!"
    >>> x println
    Hello, world!
