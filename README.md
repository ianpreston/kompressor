# Sol

Sol is a prototype language based on [Io Language](http://iolanguage.org/).

## Install & run

    $ git clone https://github.com/ianpreston/sol
    $ cd sol && virtualenv venv && source venv/bin/activate
    $ py.test sol/
    $ python main.py


## Hello world

    >>> x := "Hello, world!"
    >>> x.println
    Hello, world!
